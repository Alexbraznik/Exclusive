## Ссылка на рабочую версию

https://exclusive-omega.vercel.app/

## Установка

Склонируйте репозиторий с проектом:

`git clone https://github.com/Alexbraznik/Exclusive`

Установите зависимость Next.js:

`npm install`

## Запуск

Проект запускается командой

`npx next dev`
